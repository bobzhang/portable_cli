# Diffskill CLI

`cmd/diffskill` summarizes unified diffs into review-focused Markdown. It does
not require Git; it reads a patch from a guest-visible file or stdin and reports
changed files, line counts, risk tags, and review hints.

## CLI Help

The option reference is generated by `@argparse`, so the checked transcript
stays in sync with the command implementation.

```mooncram
$ moon -C "$TESTDIR/../.." run --target wasm cmd/diffskill -- --help
Usage: diffskill [options]

Summarize unified diffs into review-focused Markdown.

Options:
  -h, --help             Show help information.
  -V, --version          Show version information.
  -f, --file <file>      input diff file; stdin is used when omitted
  --limit <limit>        maximum changed files to list [default: 80]
  -o, --output <output>  write report to a guest-visible file
```

## Review Summary

```mooncram
$ moon -C "$TESTDIR/../.." run --target wasm cmd/diffskill -- --file tests/cram/fixtures/diff/sample.diff
# Diff Skill

- files: 3
- additions: 4
- deletions: 2
- hunks: 2
- risk: high

## Changed Files

| file | + | - | hunks | tags |
|---|---:|---:|---:|---|
| `cmd/demo/main.mbt` | 2 | 1 | 1 | code |
| `tests/cram/demo.md` | 2 | 1 | 1 | test/docs |
| `skills/demo/assets/demo.wasm` | 0 | 0 | 0 | artifact,skill |

## Review Hints

- Binary or Wasm artifacts changed; verify they were rebuilt from source.
```

## Bundled Skill Artifact

The `portable-diffskill` skill commits a release Wasm artifact, so an agent can
summarize a diff through `moonrun` without a MoonBit build step.

```mooncram
$ root="$TESTDIR/../.."; (cd "$root" && moonrun skills/portable-diffskill/assets/diffskill.wasm --file tests/cram/fixtures/diff/sample.diff)
# Diff Skill

- files: 3
- additions: 4
- deletions: 2
- hunks: 2
- risk: high

## Changed Files

| file | + | - | hunks | tags |
|---|---:|---:|---:|---|
| `cmd/demo/main.mbt` | 2 | 1 | 1 | code |
| `tests/cram/demo.md` | 2 | 1 | 1 | test/docs |
| `skills/demo/assets/demo.wasm` | 0 | 0 | 0 | artifact,skill |

## Review Hints

- Binary or Wasm artifacts changed; verify they were rebuilt from source.
```
